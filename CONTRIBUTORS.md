# Contributors

This repository is part of the **c-geni-al** research organization
(General Enhanced Neural Intelligence Artificial Learning), a small
research lab developing executable formal frameworks for cognitive AI.

The parent research program is **dreamOfkiki** — see the OSF
pre-registration at [10.17605/OSF.IO/Q6JYN](https://doi.org/10.17605/OSF.IO/Q6JYN)
and the flagship repository at
[github.com/c-geni-al/dream-of-kiki](https://github.com/c-geni-al/dream-of-kiki).

## Authorship policy

- **Byline** : publications resulting from this repository use the
  byline `nerve-wml project contributors` with Clément Saillant as corresponding
  author. Individual contributors are credited in §Acknowledgments
  per project policy.
- **No AI co-authorship** : large language models (Claude, GPT, Qwen,
  Mistral, etc.) are used as *tools* for drafting, review, and code
  generation. They are **not** listed as co-authors on any
  publication or tagged as co-author trailers in commits. If you use
  an LLM to assist your contribution, attribution to you remains —
  please flag LLM assistance in the PR description so reviewers can
  account for it.
- **External contributors** : welcome via pull requests. If your
  contribution is substantive enough to warrant authorship on a
  future publication, we will discuss explicitly at merge time and
  add you to the relevant paper's acknowledgments or byline (your
  choice).
